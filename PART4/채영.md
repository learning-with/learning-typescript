# CHPATER 14 구문 확장

타입스크립트에서 자바스크립트와 겹치는 부분은 클래스, 열거형(enum), 네임스페이스가 있음.

- 타입스크립트의 클래스는 `useDefineForClasssFelds` 옵션과 매개변수 속성을 제외하고 자바스크립트와 거의 동일하게 동작함.
- 열거형은 종종 유용한 경우가 있어서 사용하는 편이지만 네임스페이스는 거의 사용하지 않음.
- 타입스크립트는 자바스크립트의 데코레이터 decorator에 대한 experimental proposal을 채택함.

## 14.1 클래스 매개변수 속성

> ‼️ 클래스를 많이 사용하는 프로젝트나 클래스 이점을 갖는 프레임워크가 아니라면 사용하지 않는 것을 권장.

타입스크립트는 매개변수 속성을 선언하기 위한 단축 구문을 제공함.
매개변수 앞에 readonly, public, protected, private 제한자 중 하나를 배치하면 타입스크립트가 동일한 이름과 타입의 속성도 선언하도록 지시함.

```typescript

class Engineer {
    constructor(readonly area: string) {
        console.log(`i work in the ${area} area.`);
    }
}

new Engineer("mechanical").area;
```

```typescript

class NamedEngineer {
    fullName: string;

    constructor(
        name: string,
        public area: string;
    ) {
        this.fullName = `${name}, ${area} engineer`;
    }
}
```

위 코드는 매개변수 속성이 없는 클래스와 코드가 비슷해 보이지만 area를 명시적으로 할당하기 위한 코드가 몇줄 더 있음.

해당 방식은 매개변수와 매개변수 타입을 두 번 선언해야 하는 번거로움을 해결할 수 있음.


## 14.2 실험적인 데코레이터

데코레이터는 @와 함수명을 먼저 배치해 클래스와 멤버에 주석을 달 수 있도록 하는 자바스크립트 문법임.

```typescript
@myDecorator
class MyClass { /**/ };
```

`experimentalDecorators : true` 옵션 설정해서 사용하기

- 데코레이팅 하는 엔티티가 생성되자마자 한 번 실행됨.
- 타입스크립트에서 실험적인 기능임

## 14.3 열거형

연관된 값의 집합, 객체를 enum열거형이라고함. key와 value형태로 타입으로 지정할 수 있음.

### 14.3.1 자동 숫잣값

열거형은 명시적인 초기값을 가질 필요가 없음. 자바스크립트에서 사용할 때는 타입을 명시한 것처럼 보임.

```typescript
enum VisualTheme {
    Dark, // 0
    Light, // 1
    System, // 2
}
```
만약 첫 값에 숫자를 명시하면 그 뒤에 열거된 값은 순차적으로 큰 숫자를 가지게 됨

```typescript
enum Direction {
    Top = 1,
    Right, // 2
    Bottom, // 3
    Left, // 4
}
``` 
주의할 점은 데이터베이스와 같은 곳에 열거형 값을 저장하고 있다면 열거형 순서를 변경하거나 항목을 제거할 수 없음. 삭제될 경우 저장된 번호가 수정된 값과 달라지기 떄문에 데이터 손상 발생.

### 14.3.2 문자열값을 갖는 열거형

숫자대신 문자열을 값으로 사용 가능. 단, 모든 문자열로 구성된 열거형을 만들 수 있지만 이런 코드는 혼란스러울 수 있으므로 지양할 것.
